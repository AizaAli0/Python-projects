# Python Projects Collection

## Overview
This repository contains a collection of **Python mini projects and utility applications** built for practice and learning core programming concepts.

The projects focus on improving skills in **Python programming, automation, GUI development, and problem-solving**.

---

## Projects Included

### Digital Clock
A real-time digital clock that displays the current system time.

### Image Resizer
Resizes images to custom dimensions using Python image processing libraries.

### PDF Merger
Merges multiple PDF files into a single document.

### Robo Speaker
Converts text input into speech using text-to-speech functionality.

### Rock Paper Scissors Game
A simple interactive game against the computer.

### Text Editor App
A basic text editor with file open/save functionality.

### Tic Tac Toe Game
Classic 2-player game with win/draw detection logic.

### Weather App
Fetches and displays real-time weather data using API integration.
